BPA-CSIRO Workshops

Advancing bioinformatics expertise among Australian ‘omics researchers.

Welcome to the BPA-CSIRO Workshops Project on GitHub. This page provides an overview of the workshops and training modules that are maintained by the group. Each workshop repositories and its corresponding training modules are described in detail below.

Getting Started

To get started on how to reuse existing BTP workshops or developing new workshops, please visit the BTP Workshop Template page on GitHub. To know how the BTP images and instances are created and launched, please visit the BTP Orchestration page on GitHub.

Introduction to Next Generation Sequencing Hands-on Workshop

This is a practical 3 day hands-on course designed for bench biologists, PhD students or early career postdoctoral researchers with no or basic bioinformatics experience who are planning or currently using NGS approaches in their research area. It will include a half-day introduction to the command line interface.
